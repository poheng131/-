104 職缺分析專題

專題說明

本專題透過爬蟲自 104 人力銀行/Jobsdb蒐集數據分析相關職缺資訊，進行清理與視覺化，探討職缺分佈、熱門工具技能、產業類型與地區趨勢。

資料來源與取得方式

  • 資料來源： 104人力銀行、 JobsDB
  • 使用 Python 撰寫爬蟲腳本，定期蒐集關鍵字包含「數據分析」、「資料工程」等職缺
  • 進一步清洗、標準化職缺資訊（如技能欄位、產業類別等）

成員與分工

  • 單人作業

分析目的與預計提供的見解

  1. 整理全台數據分析相關職缺的地區分佈與密度
  2. 分析熱門工具與技能需求（如 Power BI、SQL、Python）=>何者最熱門?
  3. 探索企業產業別與薪資結構的差異
    4. =>選擇產業/作品集的依據
  4. 提供數據人才在求職時的參考建議
  5. 面議薪資ML預測與shap解釋
    6. =>談薪資的依據
